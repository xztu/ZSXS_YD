# 掌上忻师 - 在线阅档系统 #

该系统包含以下内容：

1. （高校招生领导组下属成员）查看考生档案
2. （高校招生领导组下属成员）查看阅档历史
3. （高校招生领导组）发送新到考生档案通知短信
4. （高校招生领导组）查看阅档进度
5. （高校招生领导组）查看招生进度
6. （高校招生领导组）查看最新招生简报
